# 山岳遭難解説動画 台本執筆ルール

このドキュメントは、霊夢と魔理沙が対話形式で山岳遭難事故を解説するYouTube動画の台本執筆ルールをまとめたものです。

## 基本フォーマット

### 行の構造
```
行番号→発話者<TAB>セリフ
```

- 各行は行番号（数字）で始まる
- 行番号の後に「→」
- 発話者の名前（霊夢、魔理沙、その他登場人物）
- タブ文字
- セリフ内容（必ず二重引用符で囲む）

### セリフがない行
- 行番号のみを記載（例：`266→`）
- 場面転換や間を表現するために使用

### セリフの分割
- 長いセリフは複数行に分割する
- 1行あたり概ね30-40文字程度を目安に
- 意味のまとまりで区切る

## 台本の基本構成

### 1. オープニング（必須）
```
魔理沙: なぁなぁ霊夢、今日はどんな話をしてくれるんだ？
霊夢: あら、魔理沙。
霊夢: 今日は[年]年に[山名]で発生した遭難事故の話をするわよ。
```

### 2. 山の基本情報
- 所在地（都道府県、市町村）
- 標高
- 選定（日本百名山、花の百名山など）
- 特徴的な呼び名（○○富士など）
- 山容や景観の説明

### 3. 山名の由来・歴史（該当する場合）
- 複数の説がある場合は併記
- 歴史的エピソード
- 伝説や昔話

### 4. 登山ルート情報
- 主要な登山口
- コースタイム（標準）
- 距離
- 標高差
- 難易度（信州山のグレーディングなど）
- 特徴的な地点の説明

### 5. 事故の解説
#### 導入
```
霊夢: さて。そろそろ本題である事故の解説を始めるわね。
魔理沙: ああ、よろしく頼む。[山名]で起きた遭難事故はどんなものだったんだ？
```

#### 時系列での展開
- 日付、時刻を明確に
- 登場人物の紹介（年齢、居住地、登山経験など）
- 登山計画の詳細
- 事故発生の経緯
- 救助活動
- 結果

#### 場面転換の表現
- 「さて。ここからは〜」
- 「時刻を少し戻し〜」
- 「一方、〜」
- 「翌日、〜」

### 6. まとめ
```
霊夢: 今回の事故では、〜
```
- 事故の概要を再度整理
- 原因の分析
- 教訓の提示
- 視聴者への注意喚起
- 亡くなった方への哀悼（該当する場合）

## キャラクター設定

### 霊夢（解説役）
#### 口調の特徴
- 語尾：「〜わよ」「〜みたいね」「〜でしょうね」「〜そうよ」「〜ようね」
- 相槌：「ええ」「そうね」「そうでしょうね」
- 呼びかけ：「魔理沙」
- 丁寧で説明的な話し方
- 女性的な表現

#### 役割
- 事故の詳細を説明する
- 山の情報を提供する
- 事実を淡々と伝える
- まとめと教訓を述べる

### 魔理沙（聞き手役）
#### 口調の特徴
- 語尾：「〜だぜ」「〜だな」「〜だなぁ」「〜だろう」「〜だろうな」
- 相槌：「ふむ」「うーむ」「へぇ」「なるほど」「ああ」
- 感嘆：「へぇ〜！」「おお！」「マジか」「えっ！？」
- 呼びかけ：「霊夢」
- カジュアルで親しみやすい話し方
- 男性的な表現

#### 役割
- 質問を投げかける
- 視聴者の疑問を代弁する
- 驚きや感想を述べる
- 話を掘り下げる

### その他の登場人物
- 遭難者や関係者の名前は匿名化（きめぇ丸、レン、こーりん、たかはし、提督など）
- セリフは最小限に
- 心の声は括弧()で表現

## 文体・表現の特徴

### 情報の提示方法
1. **数値情報は具体的に**
   - 標高：「標高2,670m」
   - 距離：「約15km」
   - 時間：「午前6時」「約7時間」
   - コースタイム：「標準コースタイムは往復で8〜9時間」

2. **出典の明示**
   - 「〜によれば」
   - 「〜が作成した」
   - 「〜の調査によると」
   - 「〜に選定されている」

3. **不確実な情報の扱い**
   - 「〜みたいね」「〜ようね」「〜そうよ」
   - 「おそらく〜」「どうやら〜」
   - 「〜という説がある」

4. **複数説の提示**
   - 「諸説あるけど」で前置き
   - 複数の由来や理由を併記

### 特殊な表現
1. **中点と読点の組み合わせ**
   - 「冷池山荘・、種池山荘・、新越山荘」
   - 列挙の際に使用

2. **カタカナの効果音**
   - 「ズボズボ」「ガチガチ」「パラパラ」など
   - 状況を生き生きと表現

3. **心理描写**
   - 括弧()で囲む
   - 例：「（うーむ、山の上は天気が悪そうだ…）」

4. **地図や図の挿入位置**
   - 空行（行番号のみの行）で示す
   - ルート説明の際に使用

### 時間経過の表現
- 「その後、〜」
- 「翌日、〜。登山○日目。」
- 「時刻は進み、〜」
- 「〜時間後、〜」
- 「午前/午後○時頃、〜」

### 場面転換
```
霊夢: さて。ここからは〜
```
- 新しいトピックへの移行を明示
- 視聴者の理解を助ける

## 情報提示のパターン

### 登山ルートの説明
1. スタート地点の標高と名称
2. 経由地点を順に説明（標高も併記）
3. 山頂到着
4. 下山ルート（ピストン or 周回）
5. コースタイムと距離のまとめ

例：
```
霊夢: まず、標高約240ｍにある登山口から入山。
霊夢: 標準コースタイムで1時間強登ると、標高約500mの「オホーツク展望」に到着。
霊夢: その後、〜を通過し、
霊夢: 標高1,661mの羅臼岳山頂に到着。
霊夢: その後、登ってきた登山道をピストンで下山する。
```

### 登場人物の紹介
- 年齢（○代、具体的な年齢）
- 居住地（都道府県）
- 登山経験・実績
- 関係性（友人、家族、山岳会など）

### 事故発生時の描写
1. 状況の説明（天候、時刻、場所）
2. セリフや心理描写
3. 結果の説明
4. 影響や次の展開

## 教訓・まとめの構成

### まとめパートの開始
```
霊夢: 今回の事故では、〜
```

### 含めるべき要素
1. **事故の概要**
   - 誰が、いつ、どこで遭難したか
   - 結果（救助された/亡くなった）

2. **原因の分析**
   - 直接的な原因
   - 背景となった要因
   - 判断ミスや装備不足など

3. **教訓**
   - 具体的な対策
   - 装備の重要性
   - 判断の重要性

4. **視聴者への呼びかけ**
   - 「〜してくださいね」
   - 「〜するようにしましょう」
   - 「〜を忘れないでください」

5. **哀悼の意（該当する場合）**
   - 「最後に、亡くなった○○の御冥福をお祈りして、今回の話を終わりとさせてもらうわ。」

## 注意事項

### 避けるべき表現
- 過度な批判や非難
- 断定的すぎる表現（不確実な情報の場合）
- 専門用語の説明なしでの使用

### 推奨される表現
- 客観的な事実の提示
- 複数の視点からの考察
- 視聴者への教育的な内容
- 敬意を持った表現

### 情報の扱い
- 遭難者のプライバシーに配慮（名前は匿名化）
- 出典が不明確な情報は「〜という話もある」など慎重に
- 事実と推測を明確に区別

## 執筆時のチェックリスト

- [ ] オープニングは定型文で始まっているか
- [ ] 霊夢と魔理沙の口調は一貫しているか
- [ ] セリフは全て二重引用符で囲まれているか
- [ ] 行番号→発話者<TAB>セリフの形式になっているか
- [ ] 長いセリフは適切に分割されているか
- [ ] 時系列は明確か
- [ ] 数値情報（標高、時間、距離など）は具体的か
- [ ] 場面転換は明示されているか
- [ ] まとめパートで教訓が提示されているか
- [ ] 亡くなった方への哀悼の意は示されているか（該当する場合）

## 参考情報源

### 山の情報
- 標高、選定：公式データ
- 登山ルート：YAMAP、ヤマレコなどの登山記録
- グレーディング：長野県「信州山のグレーディング」など

### 事故情報
- 報道記事
- 公式発表
- 関係者の証言
- 判決文（裁判があった場合）
